# components/ui — Primitivas (sin lógica de dominio)

Componentes base sobre los tokens de Manada. Las primitivas interactivas usan
**Radix UI** por debajo (a11y). Import: `@/components/ui`.

**Contenido:** layout (Container, Section, Stack/Row, Grid, Spacer) · core (Button,
IconButton, Fab, Badge, Chip, Card, Price, Rating, Avatar, Skeleton, Separator,
SectionHeading) · forms (Field, Input, Textarea, Select, Combobox, Checkbox,
Radio/RadioGroup/RadioCard, Switch, Slider, QuantitySelector) · feedback (Toast,
Alert, Banner, Tooltip, Dialog, Drawer/Sheet, Popover, Progress, EmptyState) ·
navegación (Breadcrumb, Pagination, SearchBar, Tabs).

> Catálogo y "cuándo usar cada uno": `ai-context/COMPONENT_LIBRARY.md`.
> Styleguide vivo: `/dev/components`. Cada archivo documenta su uso en el docstring.
